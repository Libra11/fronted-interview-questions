如何给自己团队的大型前端项目设计单元测试？

JavaScript · JavaScript
创建：2023-03-11T12:59:42Z · 更新：2023-03-11T13:05:14Z

基本原则和步骤

  1. 调研基本的前端测试框架有哪些， 有哪些优劣
  2. 需要分别针对 utils 静态方法测试与 UI 交互测试（2E2测试）
  3. 针对测试， 产出衡量指标与实践规范
  4. 针对项目产出最佳实践
  5. 通过 CI 工具进行测试检查、覆盖、定时回归
