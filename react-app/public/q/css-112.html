CSS 尺寸单位有哪些？

CSS · CSS
创建：2023-03-15T14:56:09Z · 更新：2023-03-15T14:56:10Z

CSS尺寸设置的单位包括：

  1. 像素（Pixel，缩写为px）：是最常用的单位，表示屏幕上的一个点，可以精确地指定元素的大小。
  1. 百分比（Percentage，缩写为%）：相对于父元素的大小，可以根据父元素的大小来设置子元素的大小。
  1. em：相对于当前元素的字体大小，用于设置字体大小时很常用。
  1. rem：相对于根元素（即html元素）的字体大小。
  1. vh/vw：相对于视口（Viewport）的高度和宽度。
  1. cm、mm、in、pt、pc等长度单位：用于打印样式表，不建议在Web开发中使用。
  1. 自定义单位：可以通过CSS的calc()函数自定义单位，比如使用“1fr”作为网格布局中的单位。

需要注意的是，不同的浏览器可能会有不同的计算方式和默认值，因此在设置尺寸时需要进行充分的测试和兼容性处理。
