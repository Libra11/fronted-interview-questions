CSS 中的 display 属性有哪些值？【热度: 593】

TOP100互联网 · CSS · TOP100互联网
创建：2024-04-13T09:14:45Z · 更新：2024-04-13T09:28:47Z

CSS 中的display属性是一个非常重要的属性，它用于设置一个元素的显示类型。这个属性决定了元素是如何显示以及与其他元素如何交互。以下是一些常见的display属性值及其含义：

  1. none：元素不会被显示。
  1. block：元素显示为块级元素，此类元素会新起一行。
  1. inline：元素不会新起一行，其宽度只占据它的内容宽度。
  1. inline-block：元素横排显示，但是同时具备块级元素的特性，比如可以设置宽高。
  1. flex：元素会变成弹性容器（flex container），其子元素会成为弹性项（flex items）。这个值允许使用弹性盒子布局（flexbox）。
  1. grid：元素会变成网格容器，其子元素会成为网格项。它开启了网格布局。
  1. table、table-row、table-cell 等：这些值让元素表现得像表格元素一样。
  1. list-item：元素会表现为列表项（像<li>元素一样）。

另外还有一些新的、较少使用或是实验性的display属性值，例如：

  • inline-flex: 使元素的内容为弹性容器，与flex相似，但是元素自身会像inline元素一样排列。
  • inline-grid: 类似于grid, 但是元素自身表现为inline级别。

CSS3 引入了更多复杂的布局模式，包括上述的flex和grid以及其他的一些属性值。根据您使用的 CSS 版本，可能还有更多其他属性值存在。这些布局模式为网页布局提供了更为强大和灵活的控制手段。
